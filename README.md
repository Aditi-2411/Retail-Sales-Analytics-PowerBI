# 📊 Retail Sales Analytics Dashboard | Power BI

An interactive **Retail Sales Analytics Dashboard** developed using **Microsoft Power BI** to analyze retail business performance across stores, products, customers, and executive-level KPIs. This project demonstrates end-to-end Business Intelligence skills including data cleaning, data modeling, DAX calculations, and dashboard development.

---

## 🚀 Project Overview

This dashboard transforms raw retail sales data into meaningful business insights through interactive visualizations and KPIs.

The project covers:

- Data Cleaning using Power Query
- Data Modeling
- DAX Measures & Calculated Columns
- Interactive Dashboard Design
- Business Insights
- Executive Reporting

---

## 🎯 Objectives

- Analyze overall business performance.
- Monitor revenue, profit, and order trends.
- Identify top-performing regions, stores, and products.
- Understand customer demographics and purchasing behavior.
- Provide executive-level insights for decision-making.

# 📁 About the Datasets

This repository contains **two datasets** for learning and practice purposes.

### 1️⃣ Retail_Dataset_Raw.xlsx

This is the original dataset containing intentionally introduced data quality issues such as:

- Missing values
- Duplicate records
- Inconsistent values
- Incorrect data types
- Data quality problems

This dataset is intended for practicing **Power Query data cleaning**.

---

### 2️⃣ Retail_Dataset_Cleaned.xlsx

This dataset contains the cleaned version after performing data transformation.

Cleaning includes:

- Removed duplicates
- Handled missing values
- Corrected data types
- Standardized categorical values
- Removed invalid records
- Prepared tables for analysis

**Note:** The Power BI (.pbix) file also contains all Power Query transformation steps, allowing users to understand the complete ETL process.

# 💡 Key Business Insights

- Revenue exceeded **93 Million**.
- Profit Margin achieved **35%**, indicating strong profitability.
- South region generated the highest revenue.
- Warehouse stores outperformed other store types.
- Consumer customers contributed the highest revenue.
- Female customers generated a larger share of revenue.
- UPI emerged as the preferred payment method.
- Top-performing products significantly contributed to overall profit.
- Most orders were successfully delivered, reflecting efficient order fulfillment.
